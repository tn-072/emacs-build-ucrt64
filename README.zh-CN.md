# emacs-build-ucrt64
尝试使用 UCRT 编译 Windows 版 Emacs。

从 [Emacs China 的讨论](https://emacs-china.org/t/windows-ucrt/27711/5) 得知，
目前 master 分支的 Emacs 31 已经支持使用 MSYS2 的 UCRT64 环境直接编译。
